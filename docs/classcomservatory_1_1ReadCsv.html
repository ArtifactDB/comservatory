◆ keep_subset

bool comservatory::ReadCsv::keep_subset = false

Whether to keep only a subset of fields. If false, keep_subset_names and keep_subset_indices are ignored.

◆ keep_subset_indices

std::vector<int> comservatory::ReadCsv::keep_subset_indices

Container of integers with the indices of the fields to retain.

Indices are expected to non-negative and less than the total number of fields. Values outside of this range will be ignored.

◆ keep_subset_names

std::vector<std::string> comservatory::ReadCsv::keep_subset_names

Container of strings with the names of the fields to retain.

Field names do not have to actually be present in the CSV file - if absent, they are simply ignored.

◆ parallel

bool comservatory::ReadCsv::parallel = false

Whether to parallelize reading and parsing in multi-threaded environments.

◆ validate_only

bool comservatory::ReadCsv::validate_only = false

Whether to only validate the CSV structure, not store any of the data in memory. If true, all settings of creator, keep_subset, keep_subset_names and keep_subset_indices are ignored.
